# Contribution Guide

Thank you for considering contributing to AI Accelerate!

## How to Contribute
1. Fork the repo
2. Create a branch for your feature (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Guidelines
- Keep the tone professional and motivational
- Use anonymized language for regulated examples
- Add Mermaid diagrams for any new architecture
- Follow the existing SKILL.md format for new skills

We welcome new patterns, skills, agents, and case studies.

Questions? Open an issue.
