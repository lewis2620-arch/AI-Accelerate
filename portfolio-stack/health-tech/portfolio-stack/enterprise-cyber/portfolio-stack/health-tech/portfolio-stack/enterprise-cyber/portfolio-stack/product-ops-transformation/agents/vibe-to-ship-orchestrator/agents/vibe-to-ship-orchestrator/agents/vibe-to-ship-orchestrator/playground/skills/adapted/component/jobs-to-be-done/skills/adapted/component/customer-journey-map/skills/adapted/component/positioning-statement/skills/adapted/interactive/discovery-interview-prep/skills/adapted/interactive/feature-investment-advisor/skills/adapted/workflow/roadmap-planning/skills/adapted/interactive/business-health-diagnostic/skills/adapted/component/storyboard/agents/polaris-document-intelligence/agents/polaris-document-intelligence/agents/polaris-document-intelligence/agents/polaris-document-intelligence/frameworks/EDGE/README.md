# EDGE Framework – Outcome-Driven Product Ops

**Purpose:** Shift teams from project delivery to measurable outcomes.

**Phases:**
- **Empathize** – Deep user insight & journey mapping
- **Define** – Problem framing, opportunity sizing, OKRs
- **Govern** – Roadmap governance, intake rituals, prioritization
- **Evolve** – Experimentation loops, iteration, instrumentation

**Agentic Upgrade:** Full multi-agent orchestration in agents/edge-govern-orchestrator/

**Regulated Example:** Used in large healthcare workflow transformations.
