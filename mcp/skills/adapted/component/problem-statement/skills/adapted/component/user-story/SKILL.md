# User Story

**Purpose:** Capture requirements in user-centric, testable format.

**Key Concepts:** As a [user], I want [goal] so that [benefit] + Gherkin acceptance criteria.

**Application:** For backlog refinement or agentic story generation.

**Product Edge Extension:** Integrates with EDGE Define and PLG triggers.
